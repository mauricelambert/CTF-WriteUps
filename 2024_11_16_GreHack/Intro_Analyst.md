# Analyst

The challenge is a log analysis, the log file format is `mJSON`, multilines JSON, so loop on line, load the stripped line as json and use data, very easy (i do it in a python console):

```python
>>> from json import loads
>>> for a in open("grehack_access.log"):
...     b = loads(a.strip())
...     if b["status"] == 200: print(b["request"]["uri"])
... 
>>> 
```
